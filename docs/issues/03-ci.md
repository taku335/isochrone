## ゴール

push / PR ごとに品質ゲート(lint / typecheck / test)を自動実行する GitHub Actions を整備する。

## 作業内容

- `.github/workflows/ci.yml`: pnpm キャッシュ付きセットアップ → `pnpm lint` → `pnpm typecheck` → `pnpm test`
- main への push と全 PR をトリガーに
- 後続 Issue(#9 データ検証、#8 サイズゲート)がジョブを追加しやすい構造にしておく

## 受入基準

- [ ] PR 作成で CI が走り、グリーンになる
- [ ] わざと lint エラーを入れたコミットで CI が fail する(確認後 revert)

## 検証方法

実際に PR を作って Actions の実行ログを確認。
