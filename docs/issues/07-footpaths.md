## ゴール

停留所間の徒歩乗換(footpaths)を事前生成し、同名ポールのグループ情報とあわせてブラウザ用データに含める。

## 作業内容

- 直線距離 300m 以内の停留所ペアを接続。所要 = `距離 / 80m/分 + 1 分`(切り上げ)
- 緯度経度の空間グリッド(セル ≈300m)で近傍セルのみ比較し O(n) 相当で生成(3,886 停留所の総当たりを避ける)
- **同名ポールは 500m まで必ず接続**(交差点対角のりば対策)。同名グループ(ユニーク名称 1,484)を stops データに含める
- パラメータ(半径 / 歩速 / バッファ / 同名上限)は設定ファイル化
- 出力は CSR(stop → [対象 stop, 所要分])

## 受入基準

- [ ] footpath が対称(A→B があれば B→A)
- [ ] 「栄」など代表的な同名ポール群が全て相互接続されている
- [ ] パラメータ変更が出力に反映される単体テスト

## 検証方法

Vitest(人工座標フィクスチャ)+ 実データで件数・対称性・同名接続を検査。
