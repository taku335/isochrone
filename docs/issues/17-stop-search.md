## ゴール

停留所名で検索して出発地を選ぶ UI を作る。同名ポール(のりば違い)は 1 件に集約して表示し、選択時は同名グループ全体を出発点集合とする。

## 作業内容

- stops データ(#8)のロードとかな / 漢字の前方一致 + 部分一致検索(1,484 ユニーク名称、クライアント内で十分高速)
- 検索結果は同名グループ単位で表示。選択で地図が該当位置へ移動しマーカー表示
- 選択結果として「stopIndex の配列(同名グループ全ポール)」を保持(#13 の複数 origin にそのまま渡す形)
- キーボード操作(上下 + Enter)対応

## 受入基準

- [ ] 「さかえ」「栄」どちらでも栄を検索できる
- [ ] 同名ポールが 1 件に集約され、選択で全ポールが origin 集合になる
- [ ] 地図移動・マーカー表示が機能する

## 検証方法

ブラウザでの手動確認 + 検索ロジックの Vitest。
