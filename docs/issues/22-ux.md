## ゴール

初回データロード・探索中・エラーの各状態を丁寧に扱い、URL で探索条件を共有できるようにして M3 を完成させる。

## 作業内容

- 初回ロード(〜1.1MB)のプログレス表示、失敗時のリトライ導線
- 探索中インジケータ(Worker からの進捗)、探索失敗・未到達時のメッセージ
- URL 状態同期: 出発地(同名グループ)・日付・時刻・表示設定をクエリパラメータへ。URL を開けば同じ結果が再現
- モバイルでの操作確認(検索パネルとマップの共存)

## 受入基準

- [ ] スロットリング環境でロード進捗が表示される
- [ ] URL コピー → 別タブで同一結果が再現される
- [ ] エラー時に白画面にならず、ユーザーに次の行動が示される

## 検証方法

ブラウザ手動確認(DevTools ネットワークスロットリング含む)。
